Etsy's PHPUnit Extensions PEAR Channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover etsy.github.com/phpunit-extensions
Listing available packages: 
pear remote-list -c etsy-phpunit-extensions
Installing a package: 
pear install etsy-phpunit-extensions/package_name
Installing a specific version/stability: 
pear install etsy-phpunit-extensions/package_name-1.0.0
pear install etsy-phpunit-extensions/package_name-beta
Receiving updates via a feed: 
http://etsy.github.com/phpunit-extensions//feed.xml
